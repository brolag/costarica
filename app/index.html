Aprende a programar

Únete a nosotros y aprende a programar. Nos centraremos en Node.js básico. Si ya sabes algo de JavaScript, eso que llevas adelantado; en caso contrario te ayudaremos.

Join us and learn coding with us. You are going to learn the basics of node.js. It's a plus if you know some JavaScript beforehand, but if not we can help you out.

A tu ritmo

Puedes pasar los retos a tu propio ritmo. Sin charlas aburridas. Todos los retos son software libre y están disponibles en nodeschool.io. Podrás seguir aprendiendo después del evento.

You will go through a set of self-paced challenges. No boring talks. All challenges are Open Source and available online on nodeschool.io. This means you can continue learning after the event.

Para todos

Creemos que programar es increíble, y queremos abrirlo a todo el mundo. Por eso esperamos que todos los participantes sean majos con los demás y que respeten el código de conducta.

We think coding is awesome and everyone should feel welcome at our event. This includes that we are expecting all participants to be awesome to each other and respect the Code of Conduct.

Preguntas? Quieres ayudar o participar?

Simplemente abre un issue en GitHub y únete a la conversación. Sin aún no estás en Github, puedes crear una cuenta gratuitamente . El taller Git-it te ayudará a iniciar.

Just open an issue on GitHub and join the discussion. If you are not on GitHub yet, create a free account and check out the Git-it workshop. It will help you to get started.
